# Card Inventory App

A full-stack trading card collection tracker built with PostgreSQL, Express, and React.

## Features

- Track baseball card inventory
- View card details (player, team, year, set)
- PostgreSQL database backend
- React frontend interface

## Tech Stack

- **Database:** PostgreSQL
- **Backend:** Node.js + Express
- **Frontend:** React

## Setup

1. Clone the repository
2. Install PostgreSQL
3. Create database named `card_inventory`
4. Install backend dependencies: `cd backend && npm install`
5. Install frontend dependencies: `cd frontend && npm install`
6. Create `.env` file in backend folder with database credentials
7. Run backend: `npm run dev` (in backend folder)
8. Run frontend: `npm start` (in frontend folder)
